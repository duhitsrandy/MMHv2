# Start web dev server (Next.js)

1. Use the Shell tool to run: `npm run dev`
2. Run it **in the background** (`block_until_ms: 0`).
3. Request permissions `full_network` and `all` if the sandbox blocks port 3000.
4. Report when the server is ready (typically `http://localhost:3000`).

Do not edit files unless the server fails to start due to a fixable error.
